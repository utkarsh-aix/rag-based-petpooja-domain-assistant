# Troubleshooting Guide

## POS not opening or slow response
- Check internet connectivity
- Restart the POS application
- Verify system hardware requirements
- Contact Petpooja support if the issue continues

## Billing or KOT not generating
- Ensure the printer is connected and configured
- Check table or order status
- Restart the POS application

## Inventory mismatch
- Verify item-wise stock configuration
- Check recent sales and wastage entries
- Sync inventory data

## Attendance not syncing
- Check biometric device connectivity
- Ensure internet access
- Refresh attendance data in the dashboard

## Payroll calculation issues
- Verify attendance and shift settings
- Check overtime and penalty rules
- Review salary configuration

## Login or access issues
- Confirm correct login credentials
- Reset password if required
- Contact support for account access issues
